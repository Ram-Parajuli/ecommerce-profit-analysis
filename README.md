# E-Commerce Profit Analysis

## Overview
E-Commerce Profitability Analysis: Identifying the root cause of profit decline despite revenue growth.

## Project Workflow
- Raw Data Collection  
- Error Log Creation (to track data changes and issues)  
- Data Cleaning Process  
- Sales Analysis  
- Profit Analysis with key financial metrics  

## Sales Analysis
Analyzed sales performance by category, city, and discount levels to identify key trends and high/low-performing segments.

## Profit Analysis
Evaluated profit trends by income statement metrics, including gross profit Margin and Net profit margin.

# E-Commerce Profitability Analysis
Data analysis project identifying profit decline root causes and recommending strategic improvements.

#  Project Overview

### Business Problem
An e-commerce electronics retailer experienced declining profitability despite maintaining revenue levels. Management needed to identify root causes and implement corrective measures.

### Objective
Analyze Q1 2026 transaction data to:
- Identify data quality issues
- Calculate accurate profitability metrics
- Determine root causes of profit pressure
- Provide actionable recommendations


# Key Findings

### Revenue Analysis
Total Revenue:**₹1,991,450**
Total Orders: 23 (1 cancelled, 1 return)
Return Rate: 4.3% (below industry average)

Profitability Metrics
Net Profit: ₹301,210
Net Profit Margin: 15.1%
Gross Profit Margin: 16.8%
Average Discount: 11.1%

### Problem Identification
Three orders lost ₹52,270 combined due to:
- Excessive discounting (30-40% on select items)
- Product returns (₹19,450 loss)
- Payment gateway fees exceeding margin on high-discount items

---

##  Strategic Recommendations

# 1. Implement Discount Controls
Action: Cap maximum discount at 15% for all electronics  


# 2. Improve Supplier Negotiations
Action: Reduce COGS from 70% to 65% through better purchasing  

# 4. Payment Collection Priority
Action: Immediate follow-up on ₹225,700 pending payments
